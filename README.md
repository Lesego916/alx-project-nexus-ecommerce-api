## Overview
This project is a scalable E-commerce backend API built as part of the ProDev Backend Engineering Program.  
It provides core e-commerce functionality including user authentication, product management, orders, and payments.

The project demonstrates real-world backend engineering practices such as secure authentication, relational database design, API documentation, and deployment readiness.

## Tech Stack
- Django & Django REST Framework
- PostgreSQL
- JWT Authentication
- Swagger / OpenAPI
- Docker (optional)
- CI/CD (planned)

## Core Features
- User registration and authentication (JWT)
- Product catalog management
- Shopping cart functionality
- Order processing
- Admin management endpoints
- Secure and documented REST API

## Authentication
JWT (JSON Web Tokens) is used for secure authentication and authorization across all protected endpoints.

## Database
PostgreSQL is used as the primary relational database, designed with normalized schemas for scalability and performance.
